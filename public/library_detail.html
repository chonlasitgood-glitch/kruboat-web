รายละเอียดโปรเจกต์

ระบบเช็คเกรดออนไลน์ (Online Grading System) คือเว็บแอปพลิเคชันที่ช่วยให้ครูสามารถประกาศผลการเรียนให้นักเรียนทราบได้ผ่านระบบออนไลน์ โดยไม่ต้องพึ่งพาระบบใหญ่ของโรงเรียน หรือเสียค่าเช่า Server ราคาแพง

ระบบทำงานโดยใช้ Google Sheets เป็นฐานข้อมูล (Database) ซึ่งครูทุกคนคุ้นเคยดีอยู่แล้ว เพียงแค่ Copy ไฟล์ Excel เกรดลงไป ระบบหน้าเว็บก็จะดึงข้อมูลไปแสดงผลให้นักเรียนดูได้ทันที

ฟีเจอร์การทำงาน

  • Secure Login: ระบบล็อกอินด้วยรหัสนักเรียน (และวันเดือนปีเกิด หรือรหัสผ่าน)
  • Privacy First: นักเรียนเห็นข้อมูลเฉพาะของตัวเองเท่านั้น ไม่เห็นของเพื่อน
  • Real-time: แก้เกรดใน Sheet ปุ๊บ หน้าเว็บเปลี่ยนปั๊บ ไม่ต้องกด Publish ใหม่
  • Customizable: สามารถเปลี่ยนโลโก้ สีธีม และข้อความต่างๆ ได้ง่ายผ่านไฟล์ Config

สิ่งที่ต้องเตรียม

  • บัญชี Gmail สำหรับสร้าง Google Sheets
  • ไฟล์รายชื่อนักเรียน (เลขที่, ชื่อ, รหัสประจำตัว, เกรดแต่ละวิชา)
