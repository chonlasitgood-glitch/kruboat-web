01

แนวคิดเชิงคำนวณ

Computational Thinking (CT)

แนวคิดเชิงคำนวณ ไม่ใช่การคิดเหมือนคอมพิวเตอร์ แต่เป็นกระบวนการแก้ปัญหาอย่างเป็นขั้นตอน เพื่อให้คอมพิวเตอร์ (หรือมนุษย์) สามารถนำไปปฏิบัติตามได้ ประกอบด้วย 4 องค์ประกอบหลัก:

1. Decomposition

การแตกปัญหาใหญ่เป็นปัญหาย่อย: แยกส่วนประกอบให้เล็กลงเพื่อให้จัดการง่ายขึ้น เช่น การแยกชิ้นส่วนจักรยานเพื่อซ่อม

2. Pattern Recognition

การหารูปแบบ: หาความเหมือนหรือความสัมพันธ์ของปัญหาที่เคยเจอมาแล้ว เพื่อใช้วิธีเดิมแก้ปัญหาใหม่

3. Abstraction

ความคิดเชิงนามธรรม: มุ่งเน้นเฉพาะข้อมูลสำคัญ ตัดรายละเอียดที่ไม่จำเป็นออก เช่น แผนที่รถไฟฟ้าที่ไม่บอกระยะทางจริงแต่บอกสถานี

4. Algorithm Design

การออกแบบขั้นตอนวิธี: ลำดับขั้นตอนการทำงานที่ชัดเจน ตั้งแต่ต้นจนจบ เช่น วิธีการทอดไข่เจียว

ตัวอย่าง: การจัดกระเป๋าไปโรงเรียน

  • Decomposition: แยกของเป็นหมวดหมู่ (หนังสือ, เครื่องเขียน, ชุดพละ)
  • Pattern Recognition: ดูตารางเรียน วันจันทร์มีวิชาอะไรบ้าง (เหมือนสัปดาห์ที่แล้ว)
  • Abstraction: สนใจแค่วิชาเรียน ไม่ต้องสนใจว่าปกหนังสือสีอะไร
  • Algorithm: 1. เปิดตารางเรียน
    2. หยิบหนังสือตามวิชา
    3. ตรวจสอบเครื่องเขียน
    4. ปิดกระเป๋า

การเขียนโปรแกรม Python

Programming & Logic Control

สัญลักษณ์ผังงาน (Flowchart) เบื้องต้น

Start/End
เริ่มต้น/สิ้นสุด
Process
การประมวลผล
Input/Output
รับ/แสดงข้อมูล
ตัดสินใจ (Decision)

โครงสร้างภาษาไพทอน

example_1.py Variables & Output 
# การประกาศตัวแปรและแสดงผล
name = "Somchai"
age = 14
height = 165.5

print("ชื่อ:", name)
print(f"อายุ {age} ปี")  # f-string
example_2.py Control Structure (If-Else) 
# การตรวจสอบเงื่อนไข
score = int(input("กรอกคะแนน: "))

if score >= 50:
    print("สอบผ่าน")
else:
    print("สอบตก")

คำสั่งพื้นฐานที่ควรรู้

  • print() คำสั่งแสดงผลข้อมูลออกทางหน้าจอ
  • input() คำสั่งรับข้อมูลจากคีย์บอร์ด (ค่าที่ได้จะเป็น String เสมอ)
  • int() / float() แปลงข้อความตัวเลขให้เป็นจำนวนเต็ม (int) หรือทศนิยม (float) เพื่อใช้คำนวณ

โครงสร้างควบคุม (Control Structures)

  • 1. การทำงานแบบลำดับ (Sequence): ทำทีละบรรทัดจากบนลงล่าง
  • 2. การทำงานแบบทางเลือก (Selection): ตัดสินใจด้วย if, elif, else
  • 3. การทำงานแบบวนซ้ำ (Repetition): ทำซ้ำด้วย for หรือ while
03

ระบบคอมพิวเตอร์

Computer Architecture & Systems

องค์ประกอบของระบบคอมพิวเตอร์

Hardware

อุปกรณ์ที่จับต้องได้ เช่น CPU, RAM, Mouse, Monitor

Software

ชุดคำสั่ง/โปรแกรม เช่น Windows, iOS, Word, Game

Peopleware

ผู้ใช้งาน/ผู้ดูแลระบบ เช่น User, Programmer, Admin

หลักการทำงาน (IPOS Cycle)

I
Input (รับข้อมูล) Keyboard, Mouse, Microphone
P
Process (ประมวลผล) CPU (สมองของคอมพิวเตอร์)
O
Output (แสดงผล) Monitor, Printer, Speaker
S
Storage (จัดเก็บข้อมูล) Hard Disk, SSD, Flash Drive (เก็บถาวร) / RAM (เก็บชั่วคราว)

รู้หรือไม่?

RAM vs ROM
- RAM: ความจำชั่วคราว ไฟดับข้อมูลหาย (ใช้ตอนเปิดโปรแกรม)
- ROM: ความจำถาวร ไฟดับข้อมูลอยู่ (เก็บชุดคำสั่งเริ่มต้นระบบ BIOS)

System vs Application Software
- System SW: ระบบปฏิบัติการ (OS) เช่น Windows, macOS, Android
- App SW: โปรแกรมประยุกต์ เช่น Chrome, Photoshop, Line

เทคโนโลยีสารสนเทศกับความปลอดภัย

Cyber Security & Ethics

ภัยคุกคามทางคอมพิวเตอร์

  • Malware (มัลแวร์): โปรแกรมประสงค์ร้าย เช่น Virus, Worm, Trojan
  • Hacker / Cracker: ผู้ที่มีความรู้สูงและเจาะระบบคอมพิวเตอร์ (Cracker เจาะเพื่อทำลาย)
  • Phishing: การหลอกลวงทางอินเทอร์เน็ตเพื่อให้กรอกข้อมูลส่วนตัว

การใช้งานอย่างมีความรับผิดชอบ

  • Privacy: ไม่เปิดเผยข้อมูลส่วนตัวของตนเองและผู้อื่น (ชื่อ, ที่อยู่, รหัสผ่าน)
  • Copyright: เคารพสิทธิความเป็นเจ้าของผลงาน ไม่คัดลอกงานผู้อื่นโดยไม่อ้างอิง
  • Etiquette: มีมารยาท ไม่ใช้ถ้อยคำรุนแรง (Cyberbullying) หรือสร้างความเกลียดชัง

แนวทางปฏิบัติเมื่อพบข้อมูลไม่เหมาะสม

1. ปฏิเสธการรับข้อมูลนั้น 2. ไม่ส่งต่อ (ไม่ Share) 3. แจ้งผู้ดูแลระบบหรือผู้ปกครอง 4. บล็อก (Block) ผู้ส่ง
