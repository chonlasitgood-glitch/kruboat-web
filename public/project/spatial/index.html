AR SPATIAL MATH
เกี่ยวกับโครงการ งานวิจัย ผู้จัดทำ
Research Project

เรียนรู้คณิตศาสตร์มิติสัมพันธ์
ด้วยเทคโนโลยี Web-based AR

นวัตกรรมการเรียนรู้ที่ผสานโลกจริงและโลกเสมือน ควบคุมวัตถุ 3 มิติด้วยมือเปล่าผ่านเว็บเบราว์เซอร์ โดยไม่ต้องติดตั้งแอปพลิเคชัน

เข้าสู่บทเรียน (นักเรียน) สร้างโจทย์ (ครู)

Core Technologies

Hand Tracking AI

ใช้ Google MediaPipe ในการตรวจจับพิกัดมือ 21 จุด แบบ Real-time เพื่อควบคุมการหมุนและตำแหน่งของวัตถุ

3D Interactive

แสดงผลกราฟิก 3 มิติด้วย Three.js จำลองโจทย์การนับลูกบาศก์ที่ซับซ้อนได้อย่างสมจริง

Cloud Database

ระบบเชื่อมต่อ Firebase Firestore ช่วยให้ครูสามารถสร้างและบันทึกโจทย์ใหม่ๆ ออนไลน์ได้ทันที

เกี่ยวกับงานวิจัย

โครงการนี้เป็นส่วนหนึ่งของการวิจัยเพื่อศึกษาผลสัมฤทธิ์ทางการเรียนวิชาคณิตศาสตร์ เรื่องมิติสัมพันธ์ (Spatial Reasoning) โดยใช้สื่อความเป็นจริงเสริมบนเว็บ (Web-based Augmented Reality)

ระบบถูกออกแบบให้ใช้งานง่าย ลดภาระทางเทคนิคของผู้ใช้ (Cognitive Load) และเพิ่มปฏิสัมพันธ์ (Interactivity) ผ่านการใช้มือควบคุมโดยตรง (Direct Manipulation) ซึ่งสอดคล้องกับทฤษฎีการเรียนรู้เชิงรุก

Technology Stack:

HTML5/JS Three.js Google MediaPipe Firebase Firestore Tailwind CSS
