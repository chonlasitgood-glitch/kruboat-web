RESEARCH ABSTRACT

งานวิจัยนี้มุ่งเน้นพัฒนาเครื่องมือส่งเสริมทักษะมิติสัมพันธ์ (Spatial Reasoning) ซึ่งเป็นทักษะพื้นฐานสำคัญทางคณิตศาสตร์และวิศวกรรมศาสตร์ โดยขจัดอุปสรรคด้านฮาร์ดแวร์และการเข้าถึง

ระบบทำงานแบบ 100% Client-Side ผ่านเว็บเบราว์เซอร์มาตรฐาน รองรับการประมวลผล AI บนอุปกรณ์ปลายทาง (Edge AI) ทำให้มีความรวดเร็ว ปลอดภัย และใช้งานได้ทันทีบนอุปกรณ์ที่หลากหลาย

Beyond the Screen
Into Reality

Hand Tracking AI

ใช้โมเดล Deep Learning ตรวจจับพิกัดมือ 21 จุด (Landmarks) แบบ Real-time เพื่อสร้างปฏิสัมพันธ์ที่แม่นยำและเป็นธรรมชาติเสมือนจับต้องวัตถุจริง

Interactive 3D Rendering

แสดงผลกราฟิกสามมิติคุณภาพสูงด้วย WebGL ที่ตอบสนองต่อการหมุน (Rotation) และการเคลื่อนที่ (Translation) ตามสรีระของมือผู้ใช้งาน

Dynamic Content System

ระบบจัดการโจทย์แบบ Cloud-based ช่วยให้ผู้สอนสามารถออกแบบ สร้าง และบันทึกชุดแบบทดสอบใหม่ๆ ได้อย่างไร้ขีดจำกัด

HOW IT WORKS

กระบวนการทำงานเบื้องหลังที่ผสานโลกจริงและโลกเสมือนเข้าด้วยกัน

01

Camera Input

ระบบรับภาพวิดีโอจากกล้อง Webcam ของผู้ใช้งานโดยตรงผ่าน Browser API

02

AI Inference

MediaPipe ทำการวิเคราะห์ภาพเพื่อระบุตำแหน่งของมือและนิ้วทั้ง 21 จุดแบบ Real-time

03

Logic & Physics

คำนวณเวกเตอร์ 3 มิติ (Quaternion) เพื่อแปลงค่าพิกัดมือให้เป็นการเคลื่อนไหวของวัตถุ

04

AR Rendering

Three.js วาดกราฟิก 3 มิติซ้อนทับลงบนภาพวิดีโอ ให้เสมือนวัตถุลอยอยู่บนมือจริง

POWERED BY

ขับเคลื่อนด้วยเทคโนโลยีเว็บมาตรฐานระดับโลก เพื่อประสิทธิภาพสูงสุด

Firebase

Cloud Firestore สำหรับจัดเก็บข้อมูลโจทย์และสถานะการเรียนรู้แบบ Real-time Database

MediaPipe

Machine Learning Solution ประสิทธิภาพสูงสำหรับการตรวจจับมือบนอุปกรณ์ Edge Device

Three.js

JavaScript 3D Library มาตรฐานสำหรับการเรนเดอร์กราฟิก WebGL บนเบราว์เซอร์

Tailwind CSS

Utility-first CSS framework สำหรับการออกแบบ User Interface ที่ทันสมัยและตอบสนองได้ดี
